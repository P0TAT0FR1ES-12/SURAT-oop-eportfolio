Midterm Assignment #1 — Introduction to Java

Date: March 5, 2026  ·  Handwritten on yellow pad paper

Instructions: Kindly read the questions carefully and write your answer in the yellow pad paper together with name, course, subject, and signature on it.

1. What is Java programming language? Discuss its key features and advantages.

Java is a high-level, object-oriented programming language developed by Sun Microsystems in 1995.
  • a. Object-Oriented – Java, being object-oriented means that it can have modular and reusable codes.
  • b. Platform Independence – Java programs can run on any operating systems with a JVM (Java Virtual Machine).
  • c. Robust and Secure – Through memory management and handling, Java can provide reliability and good security.
  • d. Open-Source Libraries – Java has plenty of libraries to extend functionality.

2. Explain the concept of platform independence in Java. How does Java achieve platform independence?

Java can run on all operating systems because of a compiler. A compiler translates high-level language (human language) that the programmer codes and then translates it into machine-level language (like binary).

3. Discuss the main components of a Java program and their roles.

The main components of a Java program are Packages, Classes, Methods, Variables, and Statements.
  • a. Package – It helps in organizing codes by grouping related classes and interfaces.
  • b. Class – It defines variables and methods, serving as a blueprint for creating objects in Java.
  • c. Method – A block of code that performs a specific task; the main() method acts as the program's starting point similar to C programming's main() function.
  • d. Variable – A memory location that stores and manipulates data.
  • e. Statement – An instruction that performs an action (e.g. computation, displaying output).

4. What are the differences between JDK, JRE, and JVM in Java?

The JDK (Java Development Kit) is the "toolbox" the developers use to create, compile, and debug Java programs; the JRE (Java Runtime Environment) is the package required to run already-built Java applications; and the JVM (Java Virtual Machine) runs the Java code.

5. Explain the differences between interpreted and compiled languages. Is Java an interpreted or compiled language?

A compiled language, once compiled expresses the instructions of the target machine which is not understandable by humans. While instructions in interpreted languages are not directly executed by the target machine, but instead, read and executed by some other program. Java is both because it is compiled to bytecode, then interpreted and JIT compiled by the JVM at runtime.

Midterm Activity #5 — Personal Expense Tracker

ExpenseTrackerSurat.java
import java.util.Scanner; public class ExpenseTrackerSurat { public static void printTitle() { System.out.println("========================="); System.out.println("Personal Expense Tracker"); System.out.println("========================="); } public static double totalExpenses(double food, double transportation, double otherExpense) { return food + transportation + otherExpense; } public static String checkExceed(double total, double budget) { if (total > budget) { return "You've exceeded your budget"; } else { return "You are within your budget"; } } public static void printSummary(double total, double budget, String status) { // Enhancement 1: Calculate Remaining Budget double remaining = total - budget; System.out.println("Expense Summary"); System.out.println("Total Expenses: " + total); System.out.println("Budget: " + budget); System.out.println("Remaining Budget: " + remaining); System.out.println("Status: " + status); } // Main Method public static void main(String[] args) { Scanner input = new Scanner(System.in); printTitle(); // Enhancement 2: User Input System.out.print("Enter food expense: "); double food = input.nextDouble(); System.out.print("Enter transportation expense: "); double transportation = input.nextDouble(); System.out.print("Enter other expenses: "); double other = input.nextDouble(); System.out.print("Enter your total budget: "); double budget = input.nextDouble(); System.out.println("\n"); double total = totalExpenses(food, transportation, other); String status = checkExceed(total, budget); printSummary(total, budget, status); } }
